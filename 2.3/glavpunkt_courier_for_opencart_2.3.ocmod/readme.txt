Требования:

Версия опенкарта: 2.3
для использования модуля вы должны быть клиентом Главпункта


Установка:

1) Скопируйте файлы модуля из папки /upload на Ваш сервер в корень сайта
	(Подробнее)
		В папке upload содержатся данные файлы - необходимо разместить их в соотвествующих директориях в корне вашего интренет магазина (./ - Корень сайта)
		./admin/language/ru-ru/extension/shipping/glavpunktcourier.php
		./admin/language/en-gb/extension/shipping/glavpunktcourier.php
		./admin/view/template/extension/shipping/glavpunktcourier.tpl
		./admin/controller/extension/shipping/glavpunktcourier.php
		./catalog/language/ru-ru/extension/shipping/glavpunktcourier.php
		./catalog/language/en-gb/extension/shipping/glavpunktcourier.php
		./catalog/model/extension/shipping/glavpunktcourier.php
		./catalog/controller/checkout/glavpunktcourier.php


   или с помощью "Установки расширений" 

2) Установите следующие модули
	Модули / Расширения
	Выбрать "Доствка"
	Активируйте модуль - Главпункт - курьерская доставка
3) Настройте модуль в соотвествии с инструкцией размещенной на странице загрузки данного модуля.
